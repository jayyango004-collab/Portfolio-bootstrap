# 🌐 Jael Portfolio – Bootstrap + Neon Glassmorphism

A responsive personal portfolio website built using **HTML5, CSS3, and Bootstrap 5**, featuring a modern **neon glassmorphism (cyberpunk-style) UI**.

This project was created as part of a web development learning journey and is designed to showcase projects, skills, and contact information in a professional layout.

---

## 🚀 Live Demo
https://jael-portfolio-bootstrap.netlify.app

---

## 🧑‍💻 Features

- Fully responsive design (mobile → desktop → 4K)
- Neon glassmorphism UI (cyberpunk theme)
- Hero section with full-screen layout and overlay
- Sticky navigation bar with active states
- About Me section with skills badges
- Projects grid (6+ real projects)
- Contact section with form + social icons
- Footer with developer attribution
- Hover animations and interactive UI effects
- Auto dark/light mode support using `prefers-color-scheme`
